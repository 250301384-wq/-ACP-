# ACP Big Data Android App

This folder wraps the existing React question bank as an Android app with Capacitor.

## What is included

- `web/`: mobile web assets generated from the root React app with relative asset paths.
- `android/`: native Android project generated by Capacitor.
- `capacitor.config.ts`: Android app id and app name.

## Common commands

```bash
npm install
npm run sync
npm run open:android
```

To build a debug APK locally after installing Android Studio / Android SDK:

```bash
npm run build:debug
```

The APK will be generated under:

```text
android/app/build/outputs/apk/debug/app-debug.apk
```

For convenience, the current debug build is also available at:

```text
dist/ACP大数据题库-debug.apk
```

## Notes

The app bundles the current question bank without deduplicating entries. Build locally with Java 17 and Android SDK 34.
